Introduo

O que so grafos

Um grafo um tipo de estrutura de dados no-tabular, composta por entidades (ns), em que os relacionamentos entre as entidades so demonstrados por arestas.

Que tipos de grafos existem?

Existem dois tipos de grafos: direcionados e no-direcionados

Em um grafo direcionado, o sentido de movimento pode ser apenas na direo das setas. Esta restrio no existe em um grafo no-direcionado

Que tipos de arestas existem?

Geralmente, as arestas de um grafo possuem pesos. O peso uma medida que descreve o custo de deslocamento de um n outro

Pensando em um exemplo de deslocamento entre cidades, o peso de uma aresta entre as cidades A e B seria a distncia em kms destas cidades

Representao em cdigo-fonte

Existem duas maneiras de representar grafos em cdigo-fonte: com listas encadeadas e com matrizes de adjacncia.

Cada implementao possui seus prs e contras:

Caracterstica Lista encadeada Matriz de adjacncias
Complexidade de acesso $O(n)$ $O(1)$
Complexidade de espao $O(n)$ $O(n^2)$

Implementao como lista encadeada

Uma implementao em lista em C de grafos a seguinte:

#include <stdio.h>
#include <stdlib.h>

struct Node {
    char *nome;
    int n_vizinhos;
    struct Node* vizinhos;
};

struct Graph {
    int n_nos;
    struct Node* nos;
};

int main() {
    struct Graph *g = (struct Graph*)malloc(sizeof(struct Graph));
    g->n_nos = 3;

    g->nos = (struct Node*) malloc(sizeof(struct Node) * 3);
    g->nos[0].nome = "Santa Maria";
    g->nos[1].nome = "Cachoeira do Sul";
    g->nos[2].nome = "Frederico Westphalen";

    g->nos[0].n_vizinhos = 2;
    g->nos[1].n_vizinhos = 1;
    g->nos[2].n_vizinhos = 1;

g->nos[0].vizinhos = (struct Node*) malloc(sizeof(struct Node) * g->nos[0].n_vizinhos);
    g->nos[1].vizinhos = (struct Node*) malloc(sizeof(struct Node) * g->nos[1].n_vizinhos);
    g->nos[2].vizinhos = (struct Node*) malloc(sizeof(struct Node) * g->nos[2].n_vizinhos);

    g->nos[0].vizinhos[0] = g->nos[1];  // cachoeira do sul
    g->nos[0].vizinhos[1] = g->nos[2];  // frederico westphalen
    g->nos[1].vizinhos[0] = g->nos[0];  // santa maria
    g->nos[2].vizinhos[0] = g->nos[0];  // santa maria

for(int i = 0; i < g->n_nos; i++) {
        printf("O no %d se chama %s e possui %d vizinhos\n", i, g->nos[i].nome, g->nos[i].n_vizinhos);
        for(int j = 0; j < g->nos[i].n_vizinhos; j++) {
            printf("\tO vizinho %d do no %s se chama %s\n", j, g->nos[i].nome, g->nos[i].vizinhos[j].nome);
        }
    }

    return 0;
}

Sada:

O no 0 se chama Santa Maria e possui 2 vizinhos
    O vizinho 0 do no Santa Maria se chama Cachoeira do Sul
    O vizinho 1 do no Santa Maria se chama Frederico Westphalen
O no 1 se chama Cachoeira do Sul e possui 1 vizinhos
    O vizinho 0 do no Cachoeira do Sul se chama Santa Maria
O no 2 se chama Frederico Westphalen e possui 1 vizinhos
    O vizinho 0 do no Frederico Westphalen se chama Santa Maria
